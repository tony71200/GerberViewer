# DesignControls

## English
### Files & Roles
- `ConfigGridControl.cs`: Generic config grid used for editable/read-only settings.
- `ImagePreviewControl.cs`: Image preview utility control.
- `OffsetPreviewControl.cs` (+ `.Designer.cs`/`.resx`): Preview matching offsets and image alignment.
- `PathCanvasControl.cs` (+ `.Designer.cs`/`.resx`): Visualizes movement paths and order layout.

---

## 中文
### 文件与作用
- `ConfigGridControl.cs`：通用配置表控件，支持可编辑/只读设置。
- `ImagePreviewControl.cs`：图像预览控件。
- `OffsetPreviewControl.cs`（含 `.Designer.cs`/`.resx`）：匹配偏移与对齐预览。
- `PathCanvasControl.cs`（含 `.Designer.cs`/`.resx`）：路径与排序布局可视化。
