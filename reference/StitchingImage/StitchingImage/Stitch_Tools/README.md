# Stitch_Tools

## English
### Structure
```
Stitch_Tools/
  DesignControls/
  Matcher/
  RobotManager/
  Utils/
```

### Responsibilities
- **DesignControls/**: UI controls for previewing paths, offsets, and configs.
- **Matcher/**: Feature matching algorithms and pair matching orchestration.
- **RobotManager/**: Order generation, stitching execution, and domain models.
- **Utils/**: Shared utilities for configs, logging, parsing, and image IO.

## 中文
### 结构
```
Stitch_Tools/
  DesignControls/
  Matcher/
  RobotManager/
  Utils/
```

### 职责
- **DesignControls/**：路径/偏移/配置等的界面控件。
- **Matcher/**：特征匹配算法与配对匹配流程。
- **RobotManager/**：排序生成、拼接执行与领域模型。
- **Utils/**：配置、日志、解析与图像 IO 等工具类。
