GerberViewer Params

MainForm

  • mainTabControl: TabControl chính chứa ba tab theo thứ tự Read Gerber, Create Gerber Sample, Align and Stitching.
  • _workflowContext: WorkflowContext dùng chung để trao đổi state giữa các tab mà không truy cập private controls.

Views/ReadGerberControl

  • toolStrip, lvLayers, canvas, statusStrip: UI đọc Gerber đã tách khỏi MainForm.
  • tsbOpen_Click, ReadGerberControl_DragEnter, ReadGerberControl_DragDrop: mở file và drag/drop Gerber.
  • tsbRender_Click, tsbFit_Click, RenderPreviewAsync: refresh preview, fit view, zoom/pan/fit qua GerberCanvas.
  • tsbExportSelected_Click, tsbExportCombined_Click, RunExport: export layer và combined PNG.
  • WorkflowContext: optional shared workflow state reference.

Views/CreateGerberSampleControl

  • btnOpenSample: chọn external raster bằng Open Sample.
  • btnOpenSample_Click: cập nhật WorkflowContext.SampleRasterPath và WorkflowContext.SampleConfig.SourceRasterPath; version 1 không consume render hiện tại từ tab Read Gerber.

Views/AlignStitchingControl

  • WorkflowContext: subscribe event Changed để đọc typed shared state.
  • RefreshContextLabels: hiển thị sample raster và last stitched output path từ WorkflowContext.

Workflow/Models/WorkflowContext

  • SampleRasterPath: đường dẫn external raster sample.
  • SampleConfig: cấu hình tạo Gerber sample.
  • ManifestPath: đường dẫn manifest workflow.
  • OutputDirectory: thư mục output chung.
  • AlignStitchConfig: cấu hình align/stitch.
  • LastStitchedOutputPath: output stitch gần nhất.
  • Changed/NotifyChanged: typed event thông báo state thay đổi.
