GerberViewer Params

MainForm

  • mainTabControl: TabControl chính chứa ba tab theo thứ tự Read Gerber, Create Gerber Sample, Align and Stitching.
  • _workflowContext: WorkflowContext dùng chung để trao đổi state giữa các tab mà không truy cập private controls.

Views/ReadGerberControl

  • toolStrip, lvLayers, canvas, statusStrip: UI đọc Gerber đã tách khỏi MainForm.
  • tsbOpen_Click, ReadGerberControl_DragEnter, ReadGerberControl_DragDrop: mở file và drag/drop Gerber.
  • tsbRender_Click, tsbFit_Click, RenderPreviewAsync: refresh preview, fit view, zoom/pan/fit qua GerberCanvas.
  • tsbExportSelected_Click, tsbExportCombined_Click, RunExport: export layer và combined PNG.
  • WorkflowContext: optional shared workflow state reference.

Views/CreateGerberSampleControl

  • btnOpenSample: chọn external raster bằng Open Sample.
  • txtSamplePath: textbox read-only hiển thị đường dẫn raster sample.
  • btnLoadSampleConfig: nạp/refresh cấu hình sample vào typed property grid.
  • btnCreateSample: chạy tạo tile sample async qua SampleTileGenerator.GenerateAsync.
  • btnCancelCreateSample: cancel crop bằng CancellationTokenSource.
  • prgCreateSample: progress bar nhận SampleCropProgress.
  • lblCreateSampleStatus: trạng thái chọn file/tạo tile/cancel/fail.
  • splitContainer: layout chính gồm preview bên trái và config bên phải.
  • sampleWindow: EWindowControl preview bên trái; worker thread không truy cập trực tiếp control này.
  • sampleConfigGrid: typed PropertyGrid chỉnh GerberSampleConfig.
  • btnOpenSample_Click: cập nhật WorkflowContext.SampleRasterPath và WorkflowContext.SampleConfig.SourceRasterPath; version 1 không consume render hiện tại từ tab Read Gerber.

Views/AlignStitchingControl

  • WorkflowContext: subscribe event Changed để đọc typed shared state.
  • RefreshContextLabels: hiển thị sample raster và last stitched output path từ WorkflowContext.
  • btnOpenImageFolder, txtImageFolder, lstCapturedImages, lblImageCount: UI chọn folder ảnh capture, list ảnh sau khi load và count expected/actual.
  • btnRunAlignStitch, btnCancelAlignStitch, prgAlignStitch: UI chạy/cancel align stitch và progress.
  • resultTabControl: gồm tab Order View và Stitched Image; Order View nhúng orderPathCanvas, Stitched Image nhúng picStitchedImage.
  • LoadCapturedImages: gọi CapturedImageLoader.Load, block execution khi manifest invalid/missing, count mismatch, duplicate key, unreadable image hoặc dimensions không hợp lệ.
  • ResolveManifestPath: ưu tiên WorkflowContext.ManifestPath, sau đó AlignStitchConfig.InputManifestPath, cuối cùng sample_manifest.json trong folder ảnh.

Workflow/Models/WorkflowContext

  • SampleRasterPath: đường dẫn external raster sample.
  • SampleConfig: cấu hình tạo Gerber sample.
  • ManifestPath: đường dẫn manifest workflow.
  • OutputDirectory: thư mục output chung.
  • AlignStitchConfig: cấu hình align/stitch.
  • LastStitchedOutputPath: output stitch gần nhất.
  • Changed/NotifyChanged: typed event thông báo state thay đổi.

GerberStitching.Core/Alignment

  • ISampleAligner: interface align sample tile, trả SampleAlignmentResult; direction transform chuẩn là Captured → Sample.
  • SampleAlignmentContext: input alignment gồm SampleTileId, SampleImage, CapturedImage, ExpectedCapturedToSampleTransform, InitialCapturedToSampleTransform và Options; ảnh gốc không bị overwrite.
  • SampleAlignmentResult: result gồm Success, CapturedToSampleTransform 3x3 CV_64F, NccScore, EccCorrelation, rotation/translation/scale, OverlapRatio, Method, PreprocessingVariant, RejectionReason, Warning và optional DiagnosticImages.
  • SampleAlignmentOptions: thresholds/config gồm NccMinScore, EccMinCorrelation, MinOverlapRatio, MaxTranslationPixels, MaxAbsRotationDeg, MinScale, MaxScale, AllowNccOnlyAcceptance, AllowEccFromExpectedWhenNccFails, PyramidLevels, EccIterations, EccEpsilon và Preprocessing.
  • PreprocessingOptions: modality-aware pipeline gồm grayscale conversion, ContrastNormalization, polarity sample/captured (PolarityMode), threshold fixed/Otsu/adaptive (ThresholdMode), Sobel/Canny/HALCON-equivalent edges (EdgePreparationMode), Gerber content mask, normalized size và diagnostic image flag.
  • ModalityAwarePreprocessor.Preprocess: tạo bản processed mới cho sample/captured, không mutate bitmap input; xuất PreprocessedAlignmentImages disposable.
  • HalconNccSampleAligner: NCC coarse aligner với hook/cache per SampleTileId tương ứng create_ncc_model/find_ncc_model, convert row/column/angle sang homography Captured → Sample, dispose cache deterministic và reject khi score nhỏ hơn NccMinScore.
  • PyramidEccSampleAligner: ECC refinement path chuẩn bị ảnh single-channel float, dùng expected/NCC init, validate correlation/translation/rotation/scale/overlap/finiteness và giữ transform 3x3 CV_64F.
  • NccThenPyramidEccSampleAligner: hybrid mặc định chạy NCC rồi ECC, accept khi cả hai pass, accept NCC-only khi geometry/threshold pass với warning NccOnlyAccepted, và optionally chạy ECC từ expected-grid init nếu NCC fail.
  • NeighborMatchAcceptance.IsAccepted: helper migration cho neighbor matcher, success chỉ khi pairResult?.Eval?.IsMatch == true; không accept chỉ vì Eval != null.
  • SampleAlignmentMethod, PolarityMode, ThresholdMode, EdgePreparationMode, ContrastNormalizationMode: enums cho method/preprocessing variant.

GerberStitching.Core/Arrangement

  • NaturalSortService: service natural-sort filename, expose SupportedExtensions tối thiểu .bmp, .png, .jpg, .jpeg, .tif, .tiff, GetNaturalKey, SortFiles.
  • NaturalSortItem: DTO gồm FilePath và Key để detect duplicate natural-sort keys.
  • CapturedImageLoader: enumerate supported files, natural-sort, validate readability/dimensions, validate sample_manifest.json, block count mismatch/duplicates, map ảnh K sang sample tile K, assign OrderIndex.
  • CapturedImageLoadResult: result gồm Succeeded, Errors, Images, ExpectedTileCount, ExpectedImageWidth, ExpectedImageHeight.

GerberViewer/Stitching/PathCanvasControl

  • PathCanvasControl: control Order View được adapt từ reference StitchingImage, namespace mới GerberViewer.Stitching.DesignControls.
  • SetCapturedImages: render các node theo row/column và order index.
  • SetData: overload tương thích arrange/traversal để reuse với graph stitching.
  • OrderNodeState: state hiển thị gồm Pending, Processing, SampleAlignOk, NeighborAlignOk, Interpolated, ExpectedOffset, Failed, Excluded.

Solution/Project Configuration

  • GerberStitching.Core: project class library mới, target .NET Framework 4.8, assembly GerberStitching.Core, root namespace GerberViewer.Stitching.
  • GerberViewer.sln: thêm project GerberStitching.Core\GerberStitching.Core.csproj với mappings Debug|Any CPU, Debug|x64, Release|Any CPU, Release|x64.
  • PlatformTarget: các configuration của GerberStitching.Core set x64 để thống nhất với OpenCV/HALCON native runtime.
  • Prefer32Bit: các configuration x64 liên quan native interop trong GerberViewer, GerberEngine, EWindowControl và GerberStitching.Core set false.

GerberStitching.Core/RobotManager/Domain

  • namespace GerberViewer.Stitching.RobotManager: namespace thống nhất cho domain/order/traversal; không dùng namespace legacy StitchingImage.*.
  • OrderMode: mode traversal gồm Zigzag, Branch, BranchDown.
  • StartCorner: corner xuất phát gồm TopLeft, TopRight, BottomLeft, BottomRight.
  • RobotMovement: hướng robot gồm Left, Right, Up, Down.
  • StartOrder: version 1 chỉ có bốn preset TopLeftRight, TopLeftDown, BottomRightLeft, BottomRightUp.
  • StartOrderResolution: DTO chứa kết quả resolve gồm StartCorner và RobotMovement.
  • StartOrderResolver.Resolve(StartOrder order): map TopLeftRight => TopLeft/Right, TopLeftDown => TopLeft/Down, BottomRightLeft => BottomRight/Left, BottomRightUp => BottomRight/Up.
  • ImageInfo: thông tin tile gồm FilePath, GroupId, ImageId, PositionId, XRobot, YRobot, Row, Column.
  • OrderOptions: cấu hình traversal gồm Mode, StartCorner, RobotMovement, NodeInterval.
  • OrderOptions.FromStartOrder(StartOrder order, OrderMode mode): tạo OrderOptions từ preset StartOrder.
  • Bounds2D: bounds robot/image gồm MinX, MinY, MaxX, MaxY.
  • Bounds2D.FromPoints(IEnumerable<ImageInfo> points): tính bounds từ tập tile.
  • NodeLinks: links traversal theo ImageId gồm HNext, VNext, Prev, Next.
  • GridCell: vị trí matrix gồm Row, Column.
  • EdgeDir: hướng cạnh gồm Horizontal, Vertical.
  • PathSegment: cạnh vẽ/duyệt gồm FromId, ToId, FromCell, ToCell, Direction.

GerberStitching.Core/RobotManager/RobotArrange

  • ArrangeComponent: kết quả arrange một component gồm Index, Items, Matrix, Bounds.
  • ArrangeBatchResult: kết quả arrange theo group gồm GroupId và danh sách Components.
  • RobotArrange.FromPhysicalMatrix(int groupId, ImageInfo[,] physicalMatrix, OrderOptions options): ordering version 1 đọc physical matrix [row,column], apply start corner để tạo matrix traversal input, không dùng legacy RobotOrderer.cs.

GerberStitching.Core/RobotManager/TraversalGraph

  • TraversalGraph: graph traversal từ matrix đã arrange, expose Mode, Matrix, LinksById, CellById, PathSegments.
  • TraversalGraph.Build(List<List<ImageInfo>> matrix, OrderOptions options): build graph từ physical-matrix arrangement và apply mode Zigzag/Branch/BranchDown.
  • BuildRows: tạo horizontal row links và zigzag chain khi mode Zigzag.
  • BuildBranchDown: tạo vertical column links và head-to-head horizontal links cho mode BranchDown.
  • EnumerateEdges(): trả danh sách cạnh traversal dạng Tuple<fromId,toId,EdgeDir>.

GerberStitching.Core/Models

  • namespace GerberViewer.Stitching.Models: namespace thống nhất cho workflow/config/report models.
  • PoseSource: nguồn pose gồm SampleAlignment, NeighborAlignment, AnchorAdjusted, Interpolated, ExpectedGridOffset, Manual, Excluded, Failed.
  • GerberSampleConfig: cấu hình sample gồm Rows, Columns, TileWidth, TileHeight, StartOrder.
  • AlignStitchConfig: cấu hình align/stitch gồm MinOverlapRatio, MaxAbsRotationDeg, EnforceRobotDirection, AllowNccOnlyAcceptance, PreviewUpdateInterval, MaxPreviewMegapixels, TiffMode, OutputPath.
  • GerberWorkflowConfig: cấu hình workflow tổng hợp gồm Sample và Alignment.

  • TiffMode: enum Auto, StandardTiff, BigTiff; output stitch chỉ dùng đuôi .tif/.tiff và auto chọn BigTIFF khi estimate vượt giới hạn standard TIFF.
  • WorkflowProgress, AlignStitchWorkflowResult, TileWorkflowState, ManualAlignmentRequest, ManualAlignmentResult, ProcessingTileReport: model workflow/report/manual alignment.

GerberStitching.Core/Stitching

GerberViewer/Workflow và GerberViewer/Views

GerberStitching.Core/Configuration/GerberSampleConfig

  • GerberSampleConfig: cấu hình tạo tile sample với defaults CropOrder=Zigzag, StartOrder=TopLeftRight, InvertImage=false, OverlapValue=60, OverlapUnit=Pixel, PreprocessMode=None, KeepAspectRatio=true, OutputFormat=Png, TileNamePattern=Sample_R{row:00}_C{col:00}_O{order:000}.

GerberStitching.Core/Imaging/SampleTileGenerator

  • SampleTileGenerator.GenerateAsync: chạy crop trong Task.Run, hỗ trợ CancellationToken và IProgress<SampleCropProgress>.
  • Preprocess: load raster và xử lý Resize, FitPad, CenterCrop trong processed-source pixel coordinate space.
  • BuildTiles: tạo physical matrix [row,column], áp dụng RobotArrange/TraversalGraph, tính crop theo OrderIndex.
  • Boundaries: rounding policy bằng boundary arrays với boundary đầu 0, boundary cuối bằng kích thước processed và tile luôn trong bounds.
  • sample_manifest.json: metadata source path/hash/DPI, original/processed dimensions, transform, rows/columns, overlap, crop/start order, resolved start corner/movement, inversion, timestamp, app/spec version và tiles.
  • sample_config.json, sample_overlay.png, tiles/*: output bắt buộc trong OutputDirectory.
  • INCOMPLETE.txt: marker khi cancel/fail để partial output được đánh dấu rõ ràng.
