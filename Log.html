GerberViewer Implementation Log

All function and variable changes made for the Online Gerber Viewer-style UI/spec implementation are listed here.

FileFunction / variableChangeReason
GerberViewer/MainForm.csPreviewDpiAdded fixed preview DPI constant at 300.Keeps preview independent from Export DPI and avoids high-resolution preview allocations.
GerberViewer/MainForm.csLargePrimitiveWarningThresholdAdded 50,000 primitive threshold.Surfaces large-scene performance status for high-resolution Gerber files.
GerberViewer/MainForm.csLoadFilesTracks primitive counts and displays Parsing/large-scene status in English.Improves logic/performance visibility and status-state clarity.
GerberViewer/MainForm.csBuildOptions(bool forPreview)Preview rendering now uses PreviewDpi; exports use the selected Export DPI.Prevents Export DPI from changing preview scale/sharpness and reduces memory for large boards.
GerberViewer/MainForm.csRenderPreviewAsyncStatus text changed to “Generating preview” and “Ready (preview transform, Export DPI independent)”.Aligns with spec state language.
GerberViewer/GerberCanvas.cs_majorGridPen, _minorGridPenAdded reusable grid pens.Creates dark CAD-like grid similar to onlinegerberviewer.com.
GerberViewer/GerberCanvas.csOnPaintDraws dark grid before image and updates empty-state text.Improves visual similarity to reference UI.
GerberViewer/GerberCanvas.csDrawGrid(Graphics g)Added helper for minor/major grid rendering.Keeps grid drawing isolated and reusable.
GerberViewer/GerberCanvas.csDispose(bool disposing)Disposes grid pens in addition to preview bitmap.Prevents GDI resource leaks.
GerberViewer/MainForm.Designer.csInitializeComponent UI labels/styleRenamed toolbar labels, constrained Export DPI choices, applied dark toolbar/sidebar and light status bar.Matches spec-required controls and visual style of onlinegerberviewer.com.
GerberViewer/MainForm.Designer.cstsbPan, tsbMeasureDistance, tsbMeasureAngle, tsbClearMeasurements, tscDpiAdded interaction-mode toolbar controls and removed the unsupported 6000 DPI choice so Export DPI is constrained to 150/300/600/1200.Implements the spec-required Pan/Inspect, distance, angle, and clear-measurement commands while keeping raster export DPI choices within FR-028.
GerberViewer/MainForm.cs_measurements, _pendingImagePoints, _pendingWorldPoints, Canvas_ImageClicked, FormatDistanceMeasurement, FormatAngleMeasurement, SetInteractionModeAdded world-coordinate measurement state and calculations for two-point distance and three-point included angle measurements.Satisfies FR-041 through FR-048 by calculating ΔX, ΔY, distance, bearing, and angle from Gerber millimeter coordinates instead of screen pixels or Export DPI.
GerberViewer/GerberCanvas.csCanvasInteractionMode, CanvasMeasurementOverlay, ImageClicked, SetMeasurements, SetPendingMeasurement, DrawMeasurementsAdded an independent vector-style GDI overlay for completed and live measurements with non-image-space text/marker sizing that remains readable while zooming and panning.Provides a DPI-independent measurement overlay for the current fallback canvas and keeps wheel zoom plus middle/right-button pan available during measurement.
GerberEngine/GerberScene.csGerberScene, GerberSceneLayerAdded a renderer-neutral scene model that snapshots layers, primitives, visibility, colors, and combined millimeter bounds without storing pixels or DPI.Creates the required shared geometry source for SVG preview and PNG raster export per FR-008/FR-033.
GerberEngine/GerberSvgRenderer.csGerberSvgRenderer.RenderAdded a self-contained SVG renderer that emits a viewBox from GerberScene.BoundsMm and serializes basic strokes, arcs, flashes, and regions as vector SVG elements.Introduces the required vector preview renderer surface per FR-014/FR-015 without adding a DPI property.
GerberEngine/GerberRasterExportRenderer.csGerberRasterExportRendererAdded an explicit export-only raster renderer that accepts GerberScene plus RasterExportOptions and delegates drawing to the existing GDI+ renderer.Separates PNG export ownership of DPI from scene/SVG preview responsibilities per FR-031.
GerberEngine/RenderOptionsV12.csSvgRenderOptions, RasterExportOptionsAdded separate option objects for SVG preview and raster export; only raster export options include Dpi.Prevents SVG preview options from depending on Export DPI.
GerberEngine/GerberEngine.csBuildScene, RenderSvg, ExportLayerPng, ExportCombinedPngExposed scene/SVG APIs and routed PNG export through GerberRasterExportRenderer.Makes the facade follow the planned parser → scene → SVG/export-renderer split while preserving existing UI calls.
GerberEngine/GerberEngine.csprojCompile itemsIncluded the new scene, SVG renderer, raster export renderer, and split options source files in the engine project.Ensures the new architecture classes are compiled with GerberEngine.
GerberEngine/GerberSvgRenderer.csGerberSvgRenderer.RenderFixed SVG layer-group string construction so the generated id attribute appends the escaped layer name with valid C# string literals.Resolves compiler errors CS1003 and CS1056 reported on the SVG group append line.
